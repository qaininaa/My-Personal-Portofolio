About me

Nama saya Karina Ghaisani, biasa dipanggil Karina. Saya adalah seorang mahasiswa semester 4 yang sedang menempuh pendidikan di Universitas Pembangunan Nasional Veteran Jakarta, Fakultas Ilmu Komputer, Program Studi S1 Sistem Informasi.

Saya memiliki minat yang besar terhadap web development dan UI/UX design. Ketertarikan saya pada web development berawal dari keinginan untuk mempelajari bagaimana sebuah website dapat dibangun dengan baik, mulai dari struktur, fungsionalitas, hingga tampilan yang menarik. Sementara itu, minat saya pada UI/UX design didorong oleh apresiasi saya terhadap pentingnya menciptakan pengalaman pengguna yang intuitif dan menyenangkan dalam sebuah produk digital.

Motivasi utama saya untuk menjadi seorang fullstack developer adalah keinginan untuk memahami seluruh aspek pengembangan aplikasi web secara komprehensif. Sebagai seorang fullstack developer, saya dapat mengombinasikan keterampilan dalam pengembangan front-end dan back-end, sehingga saya dapat terlibat dalam proses pembuatan aplikasi web dari awal hingga akhir. Hal ini memungkinkan saya untuk memiliki kontrol yang lebih besar atas kualitas dan efisiensi produk yang saya kembangkan.
